                                                   выбрать https

 22. Клонировать репозиторий XML на локальный компьютер
     -  git clone https://github.com/AlenaRybakovaQA/XML.git (ссылка скопированная из GitHub)
     
 23. Внутри локального XML создать файл “new.xml”
     - cat > new.xml
 24. Добавить файл под гит
     - git add new.xml
 25. Закоммитить файл
     - -  git commit -m "create empty file"
 26. Отправить файл на внешний GitHub репозиторий
     - git push
 27. Отредактировать содержание файла “new.xml” - написать информацию о себе (ФИО, возраст, количество домашних животных, будущая желаемая зарплата). Всё написать в формате XML
     - vim new.xml
     <?xml version="1.0" encoding="UTF-8"?>
     <person>
        <ФИО>Рыбакова Елена Игоревна</ФИО>
        <Возраст>31год</Возраст>
        <КоличествоДомашнихЖивотных>Одна кошка</КоличествоДомашнихЖивотных>
        <ЖелаемаяЗарплата>1000долл</ЖелаемаяЗарплата>
        <p1>Smth</p1>
    </person>
 28. Отправить изменения на внешний репозиторий
     - git add new.xml 
     - git commit -m "added data"
     - git push
 29. Создать файл preferences.xml
     - cat> preferences.xml
 30. В файл preferences.xml добавить информацию о своих предпочтениях (Любимый фильм, любимый сериал, любимая еда, любимое время года, сторона которую хотели бы посетить) в формате XML
     - vim preferences.xml
     файл в xml формате
 31. Создать файл sklls.xml добавить информацию о скиллах которые будут изучены на курсе в формате XML
     - cat > skills.xml
       vim skills.xml
       файл в xml формате
 32. Сделать коммит в одну строку
     - git add . && git commit -m "add changes"
 33. Отправить сразу 2 файла на внешний репозиторий
     - git push

 37. Сделать Commit changes (сохранить) изменения на веб интерфейсе
